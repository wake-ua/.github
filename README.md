# WaKe - Web and Knowledge Engineering Research Group

## Descripción

El grupo WaKe (Web and Knowledge Engineering) desarrolla líneas de investigación sobre dos temáticas generales:

### 1. Desarrollo de métodos de ingeniería del software basados en el uso de conocimiento

Investigamos métodos de ingeniería del software que utilizan diversos tipos de conocimiento para diseñar aplicaciones Web, incluyendo:
- Web mashups
- RIAs (Rich Internet Applications)

### 2. Desarrollo de métodos centrados en el usuario para Knowledge Discovery

Trabajamos en el desarrollo de métodos centrados en el usuario que permitan la generación de conocimiento (Knowledge Discovery) a partir de datos estructurados heterogéneos.

---

## Áreas de Investigación

- Ingeniería del Software basada en conocimiento
- Aplicaciones Web
- Mashups Web
- Rich Internet Applications (RIAs)
- Knowledge Discovery
- Minería de datos
- Métodos centrados en el usuario
